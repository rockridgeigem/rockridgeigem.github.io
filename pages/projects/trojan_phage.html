The Trojan Phage

Project Overview

The Trojan Phage is a project which involves using phages to deliver vector constructs to host organisms. We came up with this idea as a side proejct along with our Living Vaccine Project and think that we can use the construct from this project to deliver the proteins that will be used to deliver splicing proteins that can dissolve the lyme bacterium genome. Please take a moment to view the model that we have created below which showcases the process of delivering the vaccine. We hope that we can implement this process and make it versatile enough that it may be used with other vaccinations.
